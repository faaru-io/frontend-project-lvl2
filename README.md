## Project "Generate Difference"
Educational project of the school ["Hexlet"](https://ru.hexlet.io/?ref=231189).  
The utility for finding differences in configuration files.

## Setup

```sh
$ make install
```

## Example

```sh
$ gendiff /tmp/file1.json ../experiment/file2.json
```
